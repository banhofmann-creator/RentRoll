Fix AG Grid error #272 in frontend/src/app/master-data/properties/grid/page.tsx.

AG Grid v35 requires explicit module registration. The fix is to add this near the top of the file (after the ag-grid imports, before the component):

    import { ModuleRegistry, AllCommunityModule } from 'ag-grid-community';
    ModuleRegistry.registerModules([AllCommunityModule]);
